1200. Minimum Absolute Difference
Given an array of distinct integers arr, find all pairs of elements with the minimum absolute difference of any two elements.

Return a list of pairs in ascending order(with respect to pairs), each pair [a, b] follows
a, b are from arr
a < b
b - a equals to the minimum absolute difference of any two elements in arr
________________________________________________________

Code :
class Solution {
    public List<List<Integer>> minimumAbsDifference(int[] arr) {
        List<List<Integer>> res= new ArrayList<>();
        Arrays.sort(arr);
        int min =Integer.MAX_VALUE;

        for(int i=0;i< arr.length-1;i++){
            int diff=arr[i+1]-arr[i];
            if(diff < min){
                min =diff;
                res.clear();
                res.add(Arrays.asList(arr[i], arr[i+1]));
            }
            else if(diff == min)
                res.add(Arrays.asList(arr[i], arr[i+1]));
        }
        return res;
    }
}
_______________________________________________________

Intuition :
The minimum absolute difference between any two elements is most likely to occur between numbers that are close to each other in value. By sorting the array, we bring numerically close elements next to each other, allowing us to efficiently compare only adjacent elements instead of all possible pairs. This reduces unnecessary comparisons and guarantees correctness.
_______________________________________________________

Approach :
Sort the array in ascending order.
Initialize a variable to track the minimum absolute difference.
Traverse the sorted array and compute the difference between each adjacent pair.

If a smaller difference is found:
Update the minimum difference.
Clear the result list and add the new pair.

If the difference equals the current minimum:
Add the pair to the result list.

Return the list of all pairs with the minimum absolute difference.
_______________________________________________________

Complexity : 
Time complexity: O(nlogn)
Space complexity: O(1)
