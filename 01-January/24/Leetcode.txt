1877. Minimize Maximum Pair Sum in Array
The pair sum of a pair (a,b) is equal to a + b. The maximum pair sum is the largest pair sum in a list of pairs.

Given an array nums of even length n, pair up the elements of nums into n / 2 pairs such that:
Each element of nums is in exactly one pair, and
The maximum pair sum is minimized.

Return the minimized maximum pair sum after optimally pairing up the elements.
_________________________________________________________

Code :
class Solution {
    public int minPairSum(int[] nums) {
        int min = Integer.MAX_VALUE, max = Integer.MIN_VALUE;
        int[] freq = new int[100001];
        for(int i = 0; i < nums.length; i++) {
            if(nums[i] < min) min = nums[i];
            if(nums[i] > max) max = nums[i];
            freq[nums[i]]++;
        }
        int max_sum = 0, l = min, r = max;
        while(l <= r) {
            if(freq[l] == 0) l++;
            else if(freq[r] == 0) r--;
            else {
                max_sum = Math.max(max_sum, l + r);
                freq[l]--;
                freq[r]--;
            }
        }
        return max_sum;
    }
}
_______________________________________________________

Intuition :
To minimize the maximum pair sum, large values should be paired with small values so that no single pair becomes excessively large. By always combining the current smallest available number with the current largest available number, we balance the sums across all pairs. Tracking frequencies allows us to simulate this pairing efficiently without explicitly sorting the array.
________________________________________________________

Approach :
Count the frequency of each number in the array and determine the minimum and maximum values.

Use two pointers:
l starting from the minimum value.
r starting from the maximum value.

While l â‰¤ r:
Move l forward if its frequency is zero.
Move r backward if its frequency is zero.
Otherwise, pair l and r, update the maximum pair sum, and decrement their frequencies.

Continue until all elements are paired.
Return the maximum pair sum obtained.
________________________________________________________

Complexity :
Time complexity: O(n)
Space complexity: O(10^5)
