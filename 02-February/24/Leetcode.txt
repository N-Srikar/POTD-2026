1022. Sum of Root To Leaf Binary Numbers
You are given the root of a binary tree where each node has a value 0 or 1. Each root-to-leaf path represents a binary number starting with the most significant bit.
_______________________________________________________

Code :
class Solution {
    public int sumRootToLeaf(TreeNode root) {
        return dfs(root, 0);
    }

    public int dfs(TreeNode root, int val) {
        if (root == null) return 0;
        val = val * 2 + root.val;
        return root.left == root.right ? val : dfs(root.left, val) + dfs(root.right, val);
    }
}
_______________________________________________________

Intuition :
Each root-to-leaf path forms a binary number.
As we move from root to leaf:
The current value acts like building a binary number from left to right.
At each node, we shift the current value left (multiply by 2).
Then add the node's value (0 or 1).

When we reach a leaf node, the accumulated value represents a complete binary number.
We sum all such values from every root-to-leaf path.
______________________________________________________

Approach :
Use Depth-First Search (DFS).
Maintain a running value val that represents the binary number formed so far.

At each node:
Update:
val = val * 2 + node.val
If the node is a leaf (both children are null):
Return val.

Otherwise:
Recursively compute the sum from left and right subtrees.
Return their sum.
This ensures every root-to-leaf path contributes exactly one binary number to the final answer.
_______________________________________________________

Complexity :
Time complexity: O(n)
Space complexity: O(h)
