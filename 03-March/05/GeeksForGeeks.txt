Longest Substring with K Uniques
You are given a string s consisting only lowercase alphabets and an integer k. Your task is to find the length of the longest substring that contains exactly k distinct characters.

Note : If no such substring exists, return -1. 
_______________________________________________________

Code :
class Solution {
    public int longestKSubstr(String s, int k) {
        int[] freq = new int[26];
        int left = 0;
        int distinct = 0;
        int maxLen = -1;

        for (int right = 0; right < s.length(); right++) {
            int idx = s.charAt(right) - 'a';

            if (freq[idx] == 0) distinct++;
            freq[idx]++;

            while (distinct > k) {
                int leftIdx = s.charAt(left) - 'a';
                freq[leftIdx]--;
                if (freq[leftIdx] == 0) distinct--;
                left++;
            }

            if (distinct == k) {
                maxLen = Math.max(maxLen, right - left + 1);
            }
        }

        return maxLen;
    }
}
_______________________________________________________

Approach :
Use two pointers left and right to maintain a sliding window.
Maintain a frequency array (size 26) to store count of characters in the current window.
Keep a variable distinct to track number of unique characters in the window.

Expand the window by moving right:
Increase frequency of s[right].
If its frequency becomes 1, increment distinct.

If distinct > k, shrink the window from the left:
Decrease frequency of s[left].
If frequency becomes 0, decrement distinct.
Move left++.

If distinct == k, update maxLen = max(maxLen, right - left + 1).
Continue until the string ends.
Return maxLen (or -1 if no valid substring found).
_______________________________________________________

Complexity : 
Time Complexity : O(n)
Space Complexity : O(1)
